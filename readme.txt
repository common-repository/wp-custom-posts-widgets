== Description ==

Wordpress Custom Widgets Plugin  for displaying posts, custom post types or sticky posts with custom  options.

== Installation ==
1. Download and extract the zip archive
2. Upload `wordpress-custom-posts-widget` folder to `/wp-content/plugins/`
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Add the widget to a sidebar and configure the options as desired
